Input 输入框 ​

用于用户输入文本的基础组件，支持多种类型和功能。

基础用法 ​

通过 v-model 绑定输入框的值。

<template>
  <Input v-model="value" placeholder="请输入内容" maxlength="10" style="width: 200px;"/>
</template>

<script setup lang="ts">
import { ref } from 'vue';
import { Input } from 'asd-components';

const value = ref('');
</script>

不同尺寸 ​

通过 size 属性设置输入框的大小，支持 small、medium 和 large。



<template>
  <div>
    <Input v-model="value" size="small" placeholder="小尺寸输入框" class="mb2" style="width: 200px;" />
    <br>
    <Input v-model="value" size="medium" placeholder="中尺寸输入框" class="mb2" style="width: 200px;" />
    <br>
    <Input v-model="value" size="large" placeholder="大尺寸输入框" class="mb2" style="width: 200px;" />
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue';
import { Input } from 'asd-components';

const value = ref('');
</script>

带 前后缀\前后置文案 的输入框 ​

通过 prefix 和 suffix 插槽自定义输入框的前缀和后缀内容。 通过 prepend 和 append 插槽自定义输入框前置后置文案


Https://
.com
<template>
  <Input v-model="value" placeholder="带前后缀的输入框" style="width: 400px" class="mb2">
    <template #prefix>
      <Icon icon="fa-user" />
    </template>
    <template #suffix>
      <Icon icon="fa-house" />
    </template>
  </Input>
  <br>
  <Input v-model="testInputVal" placeholder="prepend append" style="width: 400px">
    <template #prepend>Https://</template>
    <template #append>.com</template>
  </Input>
</template>

<script setup lang="ts">
import { ref } from 'vue';
import { Input } from 'asd-components';
import { Icon } from 'asd-components';

const value = ref('');
</script>

可清空输入框 ​

通过 clearable 属性支持一键清空输入框内容。

<template>
  <Input v-model="value" clearable placeholder="可清空输入框" style="width: 260px"/>
</template>

<script setup lang="ts">
import { ref } from 'vue';
import { Input } from 'asd-components';

const value = ref('');
</script>

密码框 ​

通过 showPassword 属性支持密码框的显示和隐藏切换。

<template>
  <Input v-model="value" type="password" showPassword placeholder="密码框" style="width: 260px"/>
</template>

<script setup lang="ts">
import { ref } from 'vue';
import { Input } from 'asd-components';

const value = ref('');
</script>

文本域 ​

通过设置 type="textarea" 使用多行文本输入框。 文本域高度可以通过 rows 属性控制。

<template>
  <Input v-model="value" type="textarea" placeholder="多行文本输入框" :rows="2"/>
</template>

<script setup lang="ts">
import { ref } from 'vue';
import { Input } from 'asd-components';

const value = ref('');
</script>

Input API ​

Input Attributes ​

属性名描述类型默认值
v-model输入框的绑定值string—
size输入框的尺寸'small' | 'medium' | 'large'medium
type输入框类型'text' | 'textarea' | 'password'text
placeholder输入框占位文本string—
clearable是否支持清空booleanfalse
showPassword是否支持密码显示切换booleanfalse
disabled是否禁用输入框booleanfalse
readonly输入框是否只读booleanfalse
autofocus原生属性是否自动补全booleanfalse
formform 表单预留字段

Input Slots ​

插槽名描述
prefix输入框前缀内容
suffix输入框后缀内容
prepend输入框前置内容
append输入框后置内容

Input Events ​

事件名描述回调参数
input输入框内容变化时触发(value: string)
clear点击清空按钮时触发—
change输入框内容改变且失去焦点时触发(value: string)
focus输入框获得焦点时触发(event: FocusEvent)
blur输入框失去焦点时触发(event: FocusEvent)
update:modelValue双向绑定值更新时触发(value: string)
