NumberAnimation 数字动画 ​

NumberAnimation 组件用于实现数字的动态变化效果，常用于数据展示、统计等场景。

基础用法 ​

通过简单的 NumberAnimation 组件即可实现数字的动态变化。

0
0
<template>
  <NumberAnimation :to="value1"/>
  <NumberAnimation :to="value2" thousandsSeparator=""/>
  <Button @click="handlePlay" type="primary">开始</Button>
</template>

<script setup>
import { ref } from 'vue';
import { NumberAnimation } from 'asd-components';
import { Button } from 'asd-components';

const value1 = ref(100000);
const value2 = ref(100000.123456789);

const handlePlay = () => {
  if (value1.value || value2.value) {
    value1.value = 0;
    value2.value = 0;
  } else {
    value1.value = 10000000.123456789;
    value2.value = 10000000.123456789;
  }
};


</script>

精度 ​

通过设置fixed 的 number 数值 可以实现精度的控制

0
0
<template>
  <NumberAnimation :to="value1" :autoplay="false" ref="ref1" :fixed="2"/>
  <NumberAnimation :to="value2" thousandsSeparator="" :autoplay="false" ref="ref2" :fixed="3"/>
  <Button @click="handlePlay" type="primary">开始</Button>
</template>

<script setup>
import { ref } from 'vue';
import { NumberAnimation } from 'asd-components';
import { Button } from 'asd-components';

const value1 = ref(100000.123456789);
const value2 = ref(100000.123456789);

const ref1 = ref(null)
const ref2 = ref(null)

const handlePlay = () => {
  if (value1.value || value2.value) {
    value1.value = 0;
    value2.value = 0;
  } else {
    value1.value = 10000000.123456789;
    value2.value = 10000000.123456789;
  }
  ref1.value.startPlay()
  ref2.value.startPlay()
};

</script>

前缀后缀 ​

通过 prefix 和 suffix 属性，可以为数字添加前缀或后缀，例如货币符号或单位。

$ 0
0 元
<template>
  <NumberAnimation :to="1000" prefix="$ " suffix="" :autoplay="false" ref="ref1"/>
  <NumberAnimation :to="10000" suffix=" 元" :autoplay="false" ref="ref2"/>
  <Button @click="handlePlay" type="primary">开始</Button>
</template>

<script setup>
import { ref } from 'vue';
import { NumberAnimation } from 'asd-components';
import { Button } from 'asd-components';

const ref1 = ref(null)
const ref2 = ref(null)

const handlePlay = () => {
  ref1.value?.startPlay()
  ref2.value?.startPlay()
};

</script>

自定义样式、动画时间 ​

通过设置numberStyle 自定义样式、通过设置duration设置动画时间。

0
0
<template>
  <NumberAnimation :to="value1" :numberStyle="numberStyle"/>
  <NumberAnimation :to="value2" thousandsSeparator="" :duration="5000" @onStart="onStart" @onEnd="onEnd"/>
  <Button @click="handlePlay" type="primary">开始</Button>
</template>

<script setup>
import { ref } from 'vue';
import { NumberAnimation } from 'asd-components';
import { Button } from 'asd-components';

const value1 = ref(100000);
const value2 = ref(100000);

const handlePlay = () => {
  if (value1.value || value2.value) {
    value1.value = 0;
    value2.value = 0;
  } else {
    value1.value = 10000000;
    value2.value = 10000000;
  }
};

const numberStyle = ref({
  fontSize: '30px',
  color: '#034b4e',
  fontWeight: 'normal',
});

const onStart = () => {
  console.log('onStart');
};
const onEnd = () => {
  console.log('onEnd');
};


</script>

NumberAnimation API ​

Attributes ​

属性名描述类型默认值
from动画的起始值number | string0
to动画的结束值number | string10000
duration动画持续时间（毫秒）number3000
autoplay自定播放booleantrue
fixed小数点后保留的位数number0
prefix数字的前缀string''
suffix数字的后缀string''
numberStyle自定义数字样式CSSProperties{ fontSize: '20px',color: '#000',fontWeight: 'normal',}
transition动画的缓动函数动画效果easeInOutCubic
