# 安装

`asd-components` 是一个基于 Vue 3 的组件库，旨在解决前端开发中的常见问题。以下是安装和使用该组件库的详细步骤。

## **1. 环境要求**

在安装之前，请确保你的项目满足以下要求：

- **Node.js**：版本 >= 16
- **Vue**：版本 >= 3.2
- **Vite**：推荐使用 Vite 构建工具

## **2. 安装组件库**

使用 npm 或 pnpm 安装 `asd-components`：

### **使用 npm**
```bash
npm install asd-components
```

### **使用 pnpm**

```bash
pnpm install asd-components
```

### **使用 yarn**

```bash
yarn add asd-components
```
